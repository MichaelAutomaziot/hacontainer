## הערות חשובות

### העלאת תמונות באמצעות לינקים
- ניתן להעלות תמונות מוצר ותמונות אולם תצוגה באמצעות לינקים בקובץ CSV
- תומך בכל פורמטי התמונות: webp, jpg, png וכו׳
- מספר תמונות מופרדות ב-; (נקודה-פסיק)
- לדוגמה: `https://site.com/img1.webp;https://site.com/img2.webp`

### וריאנטים - קיבוץ מוצרים
- כל צבע/גרסה של מוצר צריכה להיות בשורה נפרדת
- השדה "וריאנט" מקבץ מוצרים שונים תחת אותו מוצר בסיסי
- לדוגמה: "כיסא גיימינג Tesla דגם 7305 בצבע לבן" ו-"כיסא גיימינג Tesla דגם 7305 בצבע אדום" ישתפו את אותו וריאנט: "כיסא גיימינג Tesla דגם 7305"

### עדכון מוצרים קיימים
- אם המוצר כבר קיים במערכת (לפי מק״ט), הנתונים יתעדכנו
- אם המוצר לא קיים, ייווצר מוצר חדש

### פורמט הקובץ
- שמור את הקובץ כ-CSV עם קידוד UTF-8
- ניתן להשתמש בפסיקים (,) או טאבים להפרדה
- אם שדה ריק - פשוט השאר אותו ריק (לא לכתוב null)
- **שורת הכותרות תדולג אוטומטית**

### פורמט מפרט טכני
המפרט הטכני הוא שדה טקסט חופשי. מומלץ להשתמש בפורמט מובנה:
```
גובה: 120 ס"מ | רוחב: 65 ס"מ | משקל: 12 ק"ג | חומר: PU עור
```

### איך לייבא
1. היכנס לדף **מוצרים** במערכת
2. לחץ על כפתור **ייבוא מרובה**
3. לחץ על **בחר קובץ CSV** ובחר את הקובץ
4. או העתק-הדבק את תוכן הקובץ לשדה הטקסט
5. לחץ **ייבא מוצרים**
